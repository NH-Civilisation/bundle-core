# Bundle: Core

Curated public domain works for the Civilisation digital library.

## Structure

```
works/
  <work-slug>/
    text.md           # Full text in markdown with optional LaTeX
    chapters/         # Optional: split by chapter
      01.md
      02.md
authors/
  <author-slug>.md    # Optional: author bios for this bundle
```

## Contributing

1. Fork this repo
2. Add works following the structure above
3. Update `bundle.json` with work metadata
4. Open a PR
